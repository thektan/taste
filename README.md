## Development

Go to `node` directory.

```bash
cd node
```

Install dependencies.

```bash
yarn
```

Start the express server.

```bash
yarn start-dev
```

Open a new terminal and run the following script to recompile on any file changes.

```bash
yarn watch
```

Visit `localhost:8080` to see the app in action.
